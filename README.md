# ROMADZ STORE Website

Website produk dengan tampilan clean, profesional, dan trusted brand menggunakan konsep outline box style.

## Fitur Utama
- Desain clean & profesional dengan outline box style
- Palette warna profesional (putih, abu, hitam, aksen warna lembut)
- Responsif untuk mobile dan desktop
- Navigasi sticky header
- Hover effects yang subtle
- UX yang optimal untuk konversi

## Struktur File
- `index.html` - Halaman utama
- `produk.html` - Halaman produk & layanan
- `style.css` - Stylesheet utama
- `script.js` - JavaScript untuk interaksi
- `README.md` - Dokumentasi ini

## Cara Menjalankan
1. Download semua file ke dalam folder yang sama
2. Buka `index.html` di browser web
3. Untuk pengembangan, gunakan live server atau buka langsung di browser

## Teknologi
- HTML5
- CSS3 (Flexbox, Grid, Variables)
- Vanilla JavaScript
- Font Awesome untuk ikon
- Google Fonts (Inter & Poppins)
